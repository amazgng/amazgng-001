# Loom Real-Time Monitoring Program — V2 UI Frozen Release

## Summary
This release provides the current frozen baseline of the Loom Real-Time Monitoring Program for QT5-based SMIT loom integration.

## Included highlights
- real-time monitoring dashboard
- Machine IP Management
- per-loom enable / disable control
- live in-place refresh
- administrative declaration workflow
- diagnostics support
- support for multiple loom protocol full-status variants
